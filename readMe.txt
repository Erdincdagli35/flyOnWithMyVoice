Read Me !

Proje yapımı bir javascript türü olan p5.js kullanılarak yapılmıştır.
Bu süreçte projenin çalıştığı sırada aldığım ekran çıktılarını localhost ile çalıştım.
Bunu sağlamak için bu teknolojiyi kullandığım için p5.js'e özel bir editör kullanmam gerektiğini öğrendim.,

Bu projeyi açabilmeniz için ise bu editörü üzerinde projeyi çalıştırılmanız gerekiyor.
Bunun için size programı basit bir şekilde nasıl kullanacağınızı anlatmaya çalışacağım.

Editörü açtığımızda "File->Open" kısmından proje'yi açabilirsiniz.
Sonrasında ise sağ üst kısımda bulunan ayarlar buttonuna tıklayıp	
	Show Sidebar -> on
	Run in Browser -> on
Seçeneklerini şeklinde yapıp projemizi çalıştırabiliriz

Projemizin çalıştığında ise machine learning kütüphanesi kullanarak eklemiş olduğumuz ses komut özelliğinden yararlanabilirsiniz

Bunun için ise sadece "Up" diye seslenmeniz yeterli

Editör kendi proje dosyamın içinde de bulunmaktadır.
